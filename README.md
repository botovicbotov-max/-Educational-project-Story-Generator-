# Генератор рассказов на Python с MySQL и Kivy

## Описание
Этот проект — интерактивный генератор рассказов, созданный с использованием Python, Kivy и базы данных MySQL. Он позволяет пользователям вводить короткие теги, получать связанные рассказы, сохранять их в историю, просматривать и управлять ими. Всё работает локально, с удобным графическим интерфейсом, который легко освоить даже новичкам.

## Основные возможности
- Ввод тега для поиска или создания нового рассказа
- Генерация рассказа по тегу, либо использование существующего
- Сохранение выбранных рассказов в историю с отметкой времени
- Просмотр истории рассказов в удобной таблице с возможностью очистки
- Копирование текста рассказа в буфер обмена
- Быстрый доступ к полному тексту выбранного рассказа

## Требования
- Python 3.8+
- Библиотека Kivy
- Библиотека mysqlclient (MySQLdb)

## Установка зависимостей
Перед запуском убедитесь, что у вас установлены все необходимые библиотеки:
```bash
pip install kivy mysqlclient
```
## Настройка базы данных
### Создайте базу данных и таблицы следующими командами:

```bash
CREATE DATABASE IF NOT EXISTS general_history CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

USE general_history;

CREATE TABLE stories (
    id INT AUTO_INCREMENT PRIMARY KEY,
    tag VARCHAR(255),
    story_text TEXT
);

CREATE TABLE history (
    id INT AUTO_INCREMENT PRIMARY KEY,
    tag VARCHAR(255),
    story_text TEXT,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```
## Настройте параметры подключения в файле main.py (или в соответствующем файле), если необходимо:
```bash
DB_HOST = 'localhost'
DB_PORT = 3306
DB_USER = 'root'
DB_PASSWORD = ''  # укажите пароль, если есть
DB_NAME = 'general_history'
```

## Запуск проекта
### Запустите приложение командой:

```bash
python main.py
```

## Использование
#### Введите короткий тег в поле и нажмите "Генерировать рассказ".
#### Если рассказ по тегу найден — он отобразится. Можно сохранить его в историю.
#### Для просмотра истории нажмите "Посмотреть историю".
#### В истории можно просматривать сохранённые рассказы, удалять их или копировать текст.
#### Для очистки всей истории нажмите "Очистить историю".
